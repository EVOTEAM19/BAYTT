# Instrucciones para el Logo de BAYTT

## Ubicación del archivo

Coloca la imagen del logo en:
```
/public/logo.png
```

## Formatos soportados

- PNG (recomendado)
- SVG
- JPG/JPEG
- WebP

## Tamaño recomendado

- Ancho: 200-400px
- Alto: proporcional
- Fondo: Transparente (PNG) o negro

## El logo se mostrará automáticamente en:

- Navbar (barra superior)
- Sidebar (panel lateral del dashboard)
- Admin Sidebar (panel lateral de admin)
- Footer (pie de página)
- Páginas de autenticación (login, register, forgot-password)

## Componente

El logo se usa a través del componente `Logo` en:
`src/components/shared/logo.tsx`

Si necesitas cambiar el tamaño o comportamiento, edita ese componente.
